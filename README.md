Keeps track of movies you've watched and allows you to search for any movie.

## Features

- Save your watched movies
- Search any movie
- Share movie links with your friends

## Technology

- Google Oauth
- Debouncing search
- Uses SSR Sveltekit
- Uses pocketbase as backend deployed on fly.io
- Svelte animations
- Typescript
